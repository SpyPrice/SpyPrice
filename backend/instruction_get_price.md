## Что делет?
Скрипт получает актуальную цену товара со страницы dns-shop.ru.  
P.S. Пока что только ДНС, про другие читай ниже
## Как это работает(кратко)
1. Запускается браузер Chromium через Playwright с постоянным профилем.
2. Выполняется переход по URL, имитация движений мыши и скролла (для обхода защиты). Без них реально можно упасть в 403 ошибку
3. Извлекается цена из HTML (сначала JSON-LD, затем селектор `.product-buy__price`). Так как страница динамическая(все данные подгружаются долго) мы должны ждать
4. При неудаче сохраняется страница для отладки.

## Настройка профиля (важно!)
- При первом запуске создаётся папка `./chrome_profile`.
- Чтобы профиль прошёл проверку Cloudflare, **ОБЯЗАТЕЛЬНО** запустите скрипт один раз с `headless=False` и подожди.
- После этого можно использовать `headless=True` для автоматической работы.

## Установка всех библиотек
```bash
pip install playwright && playwright install chromium
pip install bs4
```

Теперь подробное описание. Этот срипт обходит защиту claudeflare, которая есть 
много где(DNS, Mvideo и др). НА ВБ ЧТО ТО ДРУГОЕ. Он блочит такой схему. Пока что 
хз как это обойти но буду пытаться. При первом запуске скрипта открывается окно(
так как headless=false) и создается искусственный профиль и создаются cookie
для DNS. Эта защита не проверяет тебя второй раз, поэтому ты можешь убирать отображение
окна. Окно мы отображаем так как браузер видит, что окна нету и это же видит сайт
и мы натыкаемся на защиту claudflare. А при повторном запуске сайт видит cookie
искусственного профиля и не заставляет нас пройти через claudeflare.  

И еще, ускорить загрузку цены никак нельзя. Скрипт ждет, пока ВСЕ данные из
сайта загрузятся, будь это реклама или цена. А грузится оно в рандомном порядке.
Поэтому таймер поставлен с запасом чтобы все успело загрузиться. 
Так же сорость уменьшается из-за плохого инета и замедления инета.
