## Коротко о продукте
Проект: **сервис для отслеживания и анализа изменения цен в онлайн‑магазинах** (MVP) — сбор истории цен по URL, визуализация, базовая аналитика, по возможности уведомления.

### MVP (согласовано с ТЗ)
- **Регистрация/авторизация и профиль** (минимально).
- **Отслеживаемые товары**: URL, указание магазина/источника, имя и теги для удобства.
- **Периодический сбор** и хранение временного ряда: цена, валюта (по необходимости), время фиксации, статус доступности (если источник отдаёт — иначе `unknown`/заглушка).
- **Просмотр динамики**: карточка товара с графиком, таблица последних изменений, метрики min/max/avg и **изменение за 7 и 30 дней** (и произвольный период, если успеваем).
- **Опционально для MVP**: уведомления при снижении ниже порога или изменении на N %.
- **Мини-аналитика**: топ падений/роста за период, фильтры по тегам/магазинам; **экспорт в CSV** — по возможности к финалу.
