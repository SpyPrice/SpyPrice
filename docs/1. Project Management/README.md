# Price Tracker — сервис отслеживания цен

Это копия основного `README.md` проекта для удобной навигации внутри Obsidian (раздел Project Management).

См. также: [[Team Roles]], [[Glossary]], [[Decisions]].

## Коротко о проекте
Проект: **сервис для отслеживания и анализа изменения цен в онлайн‑магазинах** (MVP).

### MVP
- Регистрация/вход и личный кабинет
- Добавление товара по ссылке
- Сбор и сохранение истории цен
- Просмотр истории (график/таблица) и базовые метрики
- (Опционально) уведомления и аналитика
